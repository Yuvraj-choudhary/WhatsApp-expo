## Getting Started

First, install all the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm start
# or
yarn start
```

Open [http://localhost:19002/](http://localhost:19002/) with your browser and click on Run on Android or IOS Emulator.
